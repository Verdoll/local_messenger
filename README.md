login "login name" - авторизация

будучи авторизованным: 
  update - обновить данные профиля
  info - данные профиля
  dialog "login friend" - открыть историю диалога с пользователем
  send "login_friend" - отправить  сообщение пользователю
  logout - выйти из профиля
  
exit - завершить программу (после выхода из профиля)
clear_data - очистить все данные
